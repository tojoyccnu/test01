# 构建错误修复

增量修复 TypeScript 和构建错误：

1. 运行构建：`pnpm build:only`

2. 解析错误输出：
   - 按文件分组
   - 按严重程度排序

3. 对于每个错误：
   - 显示错误上下文（前后各 5 行）
   - 解释问题原因
   - 提出修复方案
   - 应用修复
   - 重新运行构建
   - 验证错误已解决

4. 遇到以下情况时停止：
   - 修复引入了新错误
   - 同一错误尝试 3 次后仍未解决
   - 用户请求暂停

5. 显示摘要：
   - 已修复的错误数量
   - 剩余的错误数量
   - 新引入的错误数量

为安全起见，每次只修复一个错误！
